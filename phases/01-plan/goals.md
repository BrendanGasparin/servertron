# Project: SERVERTRON Initial Goals

The following are the goals of Project: SERVERTRON:

- Install a hypervisor on a single-node host system (SERVERTRON-1)
- Configure ZFS storage for SERVERTRON-1
- Implement a production environment with a web server, media server, and game server (Minecraft)
- Implement a lab environment for software development, testing, learning and experimentation. Implement Kubernetes in the lab environment (for learning purposes)
- Document every stage of the project
